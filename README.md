🚀 Mapeamento de Perfil de Carreira em Tecnologia com IA

Este projeto tem como objetivo utilizar inteligência artificial para identificar o perfil profissional de um usuário na área de tecnologia e, a partir disso, gerar uma trilha de estudos personalizada.

A proposta combina engenharia de prompt + análise de perfil + recomendação de carreira, simulando o papel de um mentor de carreira.

🎯 Objetivo

Criar uma solução simples e eficiente que:

Faça perguntas estratégicas para entender o perfil do usuário
Analise respostas comportamentais e técnicas
Identifique áreas mais aderentes na tecnologia
Sugira uma trilha de estudos personalizada
Gere direcionamento prático de carreira
🧠 Como funciona

O fluxo do projeto segue 3 etapas principais:

1. Coleta de perfil

A IA faz perguntas como:

Preferência por trabalho remoto ou presencial
Afinidade com código, pessoas ou processos
Forma de aprendizado
Interesse em áreas específicas (CRM, dados, desenvolvimento, etc.)
2. Análise do perfil

Com base nas respostas, a IA identifica padrões como:

Perfil técnico, analítico ou relacional
Nível de autonomia
Capacidade de resolução de problemas
Inclinação para áreas específicas da tecnologia
3. Geração da trilha de estudos

A IA sugere:

Áreas mais indicadas (ex: CRM, Product, Dados, Desenvolvimento)
Habilidades a desenvolver
Ferramentas recomendadas
Projetos práticos para portfólio
Próximos passos estratégicos
🛠️ Tecnologias utilizadas
Inteligência Artificial (modelo de linguagem)
Engenharia de Prompt
Estruturação de fluxos conversacionais
(Opcional) Integração com interfaces como:
Formulários
Chatbots
Aplicações web
💡 Exemplo de uso

Entrada do usuário:

"Prefiro trabalhar remoto, gosto de processos, CRM, fazer projetos, ensinar pessoas e lidar com clientes."

Saída esperada:

Perfil: Analítico + Relacional
Área sugerida: CRM / Customer Experience / Product
Trilha de estudos:
Fundamentos de CRM
Automação de processos
Ferramentas como Salesforce ou HubSpot
Projeto prático de implementação
